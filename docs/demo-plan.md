# 演示计划

## 第一阶段演示路径

1. 使用 `00000000000 / Admin12345@@` 登录系统。
2. 查看首页统计。
3. 创建或调整组织树。
4. 注册一个普通用户，并由管理员审批通过。
5. 在用户管理中查看用户状态。
6. 在角色模板中查看默认权限点。
7. 在权限矩阵中给用户配置 `file:upload` 等系统级权限。
8. 使用普通用户登录。
9. 上传真实文件，选择可见范围。
10. 使用不同用户登录，验证无访问权文件不会出现在列表中。
11. 下载有访问权的文件。
12. 删除文件，进入回收站。
13. 使用有权限用户恢复文件。
14. 将文件抄送给其他用户。
15. 被抄送用户查看系统内提醒和“我收到的”。
16. 管理员查看操作日志和登录日志。

## 降级说明

- CAD 第一阶段只提供上传和下载，预览入口提示“CAD 预览服务未配置，请联系管理员”。
- OnlyOffice 第一阶段保留接口和配置项，未配置时提示后续接入。
