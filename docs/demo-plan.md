# 演示计划

## 第一阶段演示路径

1. 使用 `00000000000 / Admin12345@@` 登录系统。
2. 查看首页统计。
3. 创建或调整组织树。
4. 注册一个普通用户，并由管理员审批通过。
5. 在用户管理中查看用户状态。
6. 在角色模板中查看默认权限点。
7. 在控制台维护某科室的二级资料目录。
8. 在科室资料页新增文件入口，填写富文本文件内容并开启附件上传。
9. 使用车间用户登录，进入对应科室和文件详情页。
10. 上传附件并生成上传记录。
11. 使用车间用户查看本车间上传记录。
12. 使用科室用户查看该文件下全部车间上传记录。
13. 管理员查看操作日志和登录日志。

## 降级说明

- 当前附件只做上传和下载，不做在线编辑、审批和导出。
