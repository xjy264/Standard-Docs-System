# 需求说明

## 项目目标

建设“大同房建公寓段标准化资料管理系统”，用于统一管理 Word、Excel、PPT、PDF、CAD、图片、音频、视频、压缩包和普通附件等资料。

第一阶段按真实 MVP 实施，重点保证半个月内可演示真实文件上传、下载、搜索、权限隔离、注册审批、组织管理、用户权限矩阵、回收站、抄送提醒和操作日志。

## 一期范围

- 登录、注册、滑块验证码、管理员审批。
- 组织树、部门管理、用户管理、角色模板。
- 用户系统级权限配置和权限矩阵。
- 文件夹、文件上传、文件下载、文件列表、文件搜索。
- 文件可见范围：仅自己、本部门、本部门及下级、指定人员或部门、全段。
- 文件访问权隔离：无访问权文件不出现在列表中。
- 文件软删除、回收站恢复、彻底删除权限判断。
- 文件抄送、系统内提醒、已读未读状态。
- 首页统计、操作日志、登录日志、存储统计基础版。
- Docker Compose 部署。

## 延后范围

- OnlyOffice 完整在线编辑回写和版本冲突处理。
- CAD 在线转换预览。
- 大文件分片上传、断点续传、秒传。
- 完整文件版本管理、标签、收藏、最近编辑、最近下载。
- 知识库和 AI 对话功能。
