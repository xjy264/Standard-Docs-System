# 需求说明

## 项目目标

建设“大同房建公寓段标准化资料管理系统”，用于统一配置科室资料目录、维护文件内容，并收集车间上传的标准化资料附件。

第一阶段按真实 MVP 实施，重点保证半个月内可演示注册审批、组织管理、用户权限矩阵、科室二级目录维护、文件详情维护、车间附件填报、上传记录查看和操作日志。

## 一期范围

- 登录、注册、滑块验证码、管理员审批。
- 组织树、部门管理、用户管理、角色模板。
- 用户系统级权限配置和权限矩阵。
- 科室二级资料目录维护。
- 二级目录下文件入口维护和富文本文件内容维护。
- 车间用户按文件入口上传附件。
- 科室用户查看本科室文件入口下全部车间上传记录，车间用户只查看本车间记录。
- 首页统计、操作日志、登录日志、存储统计基础版。
- Docker Compose 部署。

## 延后范围

- 大文件分片上传、断点续传、秒传。
- 附件审批、导出、版本管理和批量处理。
- 知识库和 AI 对话功能。
