# 多级资料目录树 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将科室资料页改为支持最高五层的文件夹/文件混合目录树。

**Architecture:** 新增 `sys_doc_node` 管理目录结构，文件节点继续关联 `sys_doc_item`。后端提供树查询和节点管理接口，前端 `OrgFilesView.vue` 改为折叠树展示和管理员维护入口。

**Tech Stack:** Java 21, Spring Boot 3, MyBatis-Plus, MySQL 8, Vue 3, TypeScript, Element Plus.
